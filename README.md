# Electronics Store Management System

A team project for an electronics store web application built with Node.js, Express.js, EJS, HTML, CSS, and JavaScript.

## Overview

This project is a web-based electronics store management system that includes both user-facing features and admin functions.  
It supports store operations such as browsing products, user account handling, cart-related flows, and internal management tasks.

## Main Features

### User Features
- Home page and product-related views
- User authentication pages
- Cart-related functionality

### Admin Features
- Tutorial management
- Order management
- User/account management
- Voucher management

## Tech Stack

- Node.js
- Express.js
- EJS
- HTML
- CSS
- JavaScript

## Project Structure

- `views/` - EJS templates for user and admin pages
- `public/` - static assets, CSS, and client-side scripts
- `routes/` - route handling
- `models/` - data models and database integration
